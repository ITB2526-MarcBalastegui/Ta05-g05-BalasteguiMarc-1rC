Basic Firewall Deployment with pfSense

Overview

This project focused on deploying a pfSense firewall to secure a small virtual network and applying fundamental filtering rules to control inbound and outbound traffic.

Technologies

A pfSense firewall was deployed, configured, and tested to control network traffic and block unsafe connections.

Development

The project began by installing pfSense on a virtual machine configured with two interfaces: WAN (external) and LAN (internal). After the initial setup wizard, the firewall was updated and secured with HTTPS access. A set of rules was created to block unnecessary ports such as Telnet, FTP, and SMB from external networks.

Key Features

  • Parallel event correlation
  • Reduction of false positives
  • Real‑time monitoring

Purpose

This project teaches the fundamentals of network perimeter security and how firewalls enforce organizational policies. A pfSense firewall was deployed, configured, and tested to control network traffic and block unsafe connections.

Enhancements

  • Parallel processing pipelines
  • Latency reduction
  • Optimized memory usage
